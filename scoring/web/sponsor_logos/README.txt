Graphics files placed in this directory will be detected and used as sponsor
logos in various places (scoreboard, playoffs, welcome screen.

On the welcome screen the images are sorted by filename, so if you want certain 
sponsors first it's best to prefix them with 2 digits 01, 02, 03, etc.

Recognized file extensions include: png, jpg, jpeg, gif.

To display well, a logo graphic should be approximately 150-200 pixels wide and
45-60 pixels high.
